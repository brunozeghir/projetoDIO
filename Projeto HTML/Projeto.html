Inspetor de Elementos

Em resumo, é uma ferramenta disponível no navegador onde é possível visualizar a estrutura da página em HTML, css e outras informações. É possível fazer alterações momentâneas em que após atualizarmos a página, ela voltará para sua forma original. Podemos assim, ter informações da página e até estudar com sua estrutura, tirar duvidas sobre cores, etc... Segundo o site hostgator:

A ferramenta de inspecionar elemento é um utilitário que permite visualizar o código-fonte subjacente de qualquer página web. Além disso, você pode usá-la para fazer alterações temporárias e ver os resultados em tempo real, deixando o código-fonte original intacto. Isso é extremamente útil se você precisa testar uma alteração ou diagnosticar um problema. Também pode ser de grande valia se você encontrar um site com um recurso de que goste e estiver curioso para saber como ele foi implementado.

Estrutura básica do HTML

Nesta aula aprendemos a estrutura básica do HTML.

  • Tag HTML

    • Aqui temos a Tag responsável por delimitar o documento HTML. Esta tag deverá ser aberta e fechada e tudo que estiver dentro fará parte do arquivo HTML.

  • Tag Head

    • Aqui temos a tag responsável por pré carregar informações antes de exibir para o usuário. Contém informações do comportamento do site que foi desenvolvido. Esta tag deverá ser aberta e fechada.

  • Tag Body

    • Aqui temos a tag responsável pelo corpo do site, tudo que está presente dentro desta tag irá aparecer para o usuário. Esta tag deverá ser aberta e fechada.

  • Tag Title

    • Aqui temos a tag responsável por alterar o título da página, que irá aparecer na aba do navegador. Esta tag ficará localizada dentro do head e deverá ser aberta e fechada.

Falandro sobre tags

Nesta aula, falamos sobre o que é um atributo de uma tag e conhecemos novas tags, são elas:

  • Tag i

    • Aqui temos a tag responsável por colocar o texto que estiver dentro dela em itálico. Esta tag necessita ser aberta e fechada.

  • Tag strong

    • Aqui temos a tag responsável por colocar o texto que estiver dentro dela em negrito. Esta tag necessita ser aberta e fechada.

  • Tag imput

    • Aqui temos a tag responsável por criar um campo no site para que o usuário possa preencher, seja ele um campo para preenchimento de número, letras ou senhas. Possuí atributos que distinguem as características deste campo. Esta tag é aberta e fechada na mesma tag.

  • Tag img

    • Aqui temos a tag responsável por inserir uma imagem em nosso site, através desta tag e um atributo que irá direcionar o local da imagem. A imagem pode estar localizada dentro do servidor ou pode ser um link de uma página da internet. Esta tag se fecha por sí só, não sendo necessário existir uma tag final para fechar.

  • Atributos

    • É uma configuração responsável por alterar as características de uma tag.

Atributos das tags

Nesta aula, falamos mais sobre o que é um atributo de uma tag, alguns atributos mais genéricos e outros mais específicos de uma tag.

  • ID

    • Aqui temos um atributo generalista, onde é usado para identificar uma tag. Assim, tanto no própio arquivo HTML, como no Javascript, você poderá achar um elemento através do identificador.

  • Script

    • O atributo Script serve para colocarmos comandos de estilo diretamente no elemento em que ele foi atribuído.

  • Class

    • Atributo usado também para identificar uma classe, usado com o CSS, onde identificado no CSS, você poderá atribuir um estilo específico para essa classe.

  • Type

    • Atributo específico do input, onde o seu valor altera o campo que foi criado na página. Ex: Text, recebe texto. Number, recebe somente número.

  • Src

    • Atributo utilizado na tag img. Este atributo serve para colocarmos como valor o endereço da imagem que será inserida no site. Seja o endereço de uma pasta local ou um link de uma página na internet.

  • Width

    • Atributo utilizado para determinar a largura. Aqui podemos por exemplo determinar a largura de uma imagem que inserirmos em nosso código.

Tags de textos

Aqui falamos de algumas tags mais específicas, mais voltadas para textos.

  • Tags H

    • Aqui teremos seis tags, indo desde o h1 até h6. Estas tags geralmente são usadas para criarmos títulos, subtítulos e assim por diante. Conforme o H vai subindo de número, o tamanho da fonte irá diminuindo.

  • Tag P

    • Esta tag é usada para escrevermos um novo parágrafo de um texto.

  • Blockquote

    • Tag usada para fazermos uma citação em um texto. Por padrão o texto inserido dentro desta tag é deslocado e destacado. Assim se diferenciado da tag p por exemplo. O espaçamento pode ser alterado através do CSS.

  • Tag u

    • Esta tag é usada para sublinhar o texto que estiver dentro dela.

  • Tag mark

    • Esta tag é usada para marcar um texto. Ela marca o texto selecionado pelo autor, assim destacando aquele texto.

  • Tag sup

    • Tag usada para criarmos um texto sobrescrito, como asteriscos em palavras, números, ou qualquer outro texto.

Listas Ordenadas e Não Ordenadas

Nesta aula falamos sobre listas. Aprendemos como criarmos listas ordenadas e listas não ordenadas.

  • Tag ul

    • Aqui é criado uma lista com itens não ordenados, ou seja, contendo marcadores para listar estes itens. Estes marcadores podem ser alterados via CSS.

  • Tag ol

    • Nesta tag, os itens serão ordenados através de números, seguindo uma ordem.

  • Tag il

    • Esta tag é usada para escrevermos os itens listados dentro de uma lista ordenada ou não ordenada. Ela é obrigatória dentro da OL ou UL.

Links

Nesta aula falaremos sobre links, onde aprendemos a criar itens em nosso site que poderão levar o usuário a outras páginas.

Para criarmos links, usaremos a tag A, seguida do atributo específico href. Neste atribuito iremos inserir o endereço do link. Seja através de um link de uma página externa, um arquivo html interno ou através de um ID de uma parte interna de nossa página.

Entre a abertura e o fechamento da tag, iremos digitar o texto que o usuário irá ver. Neste texto que o usuário irá clicar para leva-lo até o link que colocamos.

Por padrão, o link irá aparecer através do texto na forma azul e sublinhado. Padrão que poderá ser alterado via CSS.

Temos também o atributo target que possue dois valores importantes no uso dos links. Que são os blank e o self que são usados da seguinte forma.

Blank: Usado para abrirmos o link em outra aba do navegador ao clicar.

Self: O usuário ao clicar, será direcionado ao link na mesma aba em que está navegando.

Outro atributo interessante de ser usado é o title. Com ele é criado um tipo de balão com o texto que inserirmos no valor do atributo, quando o usuário passar o mouse em cima do link.
